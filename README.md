# kalshi-trading
System for Trading S&amp;P 500 Daily Brackets on Kalshi Prediction Markets

This script aims to identify the most optimal contract for the corresponding market available on [Kalshi.com](https://kalshi.com/)

The instructions and methodology can be found in the post - "Prediction Markets Are Literally Free Money." @ [The Quant's Playbook](https://quantgalore.substack.com/)
